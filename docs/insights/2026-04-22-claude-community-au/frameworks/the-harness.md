# The Harness thesis

Owner: Stephen Coleman
Context: Stratum (AI advisory), 6 months in market

## The claim

The model is no longer where the marginal value comes from. Model
improvements are now incremental. The exponential improvements are
happening in the **harness** — Claude Code, Claude Co-work, MCP
connectors, plugin systems, the context layer.

Implication: the skill that matters now is harness fluency, not prompt
craft.

## The TMNT-Krang metaphor

Four layers, each useless without the others:

- **Brain (Krang)** — the model. Capable but disembodied. Can't act.
- **Robot body** — the harness. Claude Code, Claude Co-work, desktop
  agents. The body that makes the brain able to do anything.
- **Weapons** — tools. MCPs, data connections, skills, hooks,
  automations.
- **Pilot** — context. Prompts, `CLAUDE.md`, skills, memory. What the
  professional user brings.

This is the mental model behind the site's L1–L5 framework.

## Interface evolution

1. **Fat apps** — Excel, SAP, desktop CRM. Logic + UI + data locked in
   one binary.
2. **SaaS** — everything in the browser. Logic + UI as a service, data
   accessed through APIs.
3. **GaaS — harness as interface.** Claude is the UI. Work happens
   inside the harness. Systems connect in via MCP/API. The dashboard
   dies; the data layer matters.

Signal: Salesforce announced a headless CRM. The interface layer is
deprecating even at the top of the market.

## Consulting implication

> "The days of building dashboards being the thing clients are looking
> for are going to decrease."

Flip the pitch: stop selling dashboards; sell MCP-accessible data
layers that a client's Claude (or ChatGPT, or in-house agent) can
query. The value moves to the pipe, not the page.

## Adoption reality

Stephen's dot visualisation: each dot = 3.2M people. The slice actually
using Claude Code or equivalent is microscopic. The rest of the market
is roughly at zero.

For L1/L2 readers this is the single most reassuring message of the
night: *you are not late.*

## How to use it on claudecode.studio

- Use "the value is in the harness" as the site's one-line thesis.
- Lead the homepage with the adoption dot chart equivalent — reassure
