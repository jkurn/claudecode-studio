## What this folder is

A structured extraction of the webinar into reusable pieces:

- `index.md` — single-file canonical extraction (read this first)
- `speakers/` — one file per speaker with bio, talk, quotes, takeaways
- `frameworks/` — standalone reference docs for each named framework
- `quotes.md` — pull-quote bank
- `tools.md` — tools & tech mentioned, one-liners, where to read more
- `newsletter-drops/` — insights mapped to the five fluency levels (L1–L5)
- `content-roadmap.md` — how to turn this into content on claudecode.studio
- `action-items.md` — concrete tasks that came out of the webinar

## Why it's organised this way

The site's thesis is that fluency grows in stages (L1 Prompter → L5 Orchestrator)
and the weekly newsletter ships "one use case, one skill, one challenge" per
level. Every insight in the webinar has a natural home in that structure;
this folder pre-sorts them so the newsletter editor can pull a drop together
in minutes instead of rewatching a two-hour talk.
