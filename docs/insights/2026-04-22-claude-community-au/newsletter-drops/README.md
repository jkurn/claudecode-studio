## Drop template

Each drop has:

- **Headline** — short, specific, promises a level-up.
- **Why this matters now** — one paragraph tied to a current signal
  (new Claude release, tool launch, industry move).
- **Use case** — a real-world scenario where this applies.
- **Skill** — a concrete thing to install, install, or configure.
- **Challenge** — a specific task the reader ships by end of week.
- **Receipt** — what "done" looks like (a commit, a screenshot, a
  shared link).

Keep each drop under ~350 words outside the code snippets.
