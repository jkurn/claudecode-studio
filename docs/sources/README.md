# Source texts

Primary source material referenced throughout `docs/insights/` and by the
Claude Code subagents in `.claude/agents/`.

- `2026-04_wei-li-practical-guide-to-gtm-autonomy.txt` — Wei Li, *The
  Practical Guide to GTM Autonomy* (April 2026). Introduces the
  Capability/Process split, four autonomy levels (Assisted / Copilot /
  Autopilot / Self-Driving), and the four rings of ownership.

- `2026-04_aidan-morgan-bankwest-aitai-slides.txt` — Aidan Morgan,
  Bankwest & AITAI slide deck (April 2026). Talk title: "Stop building
  software like it's April 2026." Source for the formal three-loop
  model (coding / backpressure / experimental deployment) and *Fitness
  Function Driven Development*.

These are plain-text extractions of the original PDFs. When quoting,
prefer this folder over summaries in `docs/insights/` — these are the
actual words spoken/written.

The webinar itself (2026-04-22 Claude Community Australia livestream)
does not have a primary transcript in the repo; the extraction under
`docs/insights/2026-04-22-claude-community-au/` is the working canonical
reference, derived from field-notes forwarded by the author.
